# Product Listing Page

[![CircleCI](https://img.shields.io/circleci/project/github/smokku/plp-demo/master.svg)](https://circleci.com/gh/smokku/plp-demo)

Demo application utilizing:

- Webpack
- Babel (ES6 + stage-2)
- Flow
- React
- Redux
- Reselect
- SASS
- CSS Modules
- UIKit
- React Flip Move
- SVG images
- SVG filters (used to colorize images)
